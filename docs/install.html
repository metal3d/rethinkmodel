Installation¶

You can install RethinkModel package with traditional manager like pip, easy_install or the prefered method pipenv. RethinkModel can of course be install in a virtualenv.

# with pipenv, in a project directory
pipenv install rethinkmodel

# with pip, with or without virtualenv
pip install rethinkmodel
