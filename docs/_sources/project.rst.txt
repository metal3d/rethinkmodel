Source code
-----------

You can fork the project on `the project Github page of Rethink:Model <https://github.com/metal3d/rethinkmodel>`_. The project page is also used to crate issues.

If you find bug or if you want to propose enhancement, fixes, or features, please:

- fork the project
- create a branch
- create a pull request

There is no guarantee that an answer will be given quickly, but we do our best to process requests. Bugs are, of course, given priority over new features.
